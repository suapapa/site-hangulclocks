# HangulClocks: 여러분의 한글시계 (Modernized)

한국의 아름다운 한글을 활용한 다양한 시계 프로젝트들을 모아놓은 갤러리/쇼케이스 웹사이트입니다.

*   **배포 URL**: [https://suapapa.github.io/site-hangulclocks](https://suapapa.github.io/site-hangulclocks)

## 🎨 디자인 에센스 (Design Essence)

이 프로젝트는 전통적인 아름다움과 현대적인 웹 기술의 조화를 지향합니다.

*   **Style**: Modern Minimalism + Cultural Elegance (Glassmorphism 적용)
*   **Palette**: 깊은 다크 테마(Deep Charcoal)와 골드 포인트(Antique Gold)
*   **Typography**: Noto Serif KR & Noto Sans KR

## 🛠 기술 스택 (Tech Stack)

*   **Frontend**: Vue 3 (Composition API)
*   **UI Framework**: Vuetify 4
*   **Build Tool**: Vite 8

## 🚀 새 시계 등록하기 (How to Register)

본인이 직접 만든 한글시계나 온라인상에 있는 멋진 한글시계 프로젝트를 제안해 주세요.

1.  `src/assets/hangulclocks.yaml` 파일에 새 정보를 추가하십시오.
2.  이후 PR(Pull Request) 또는 이슈를 통해 등록을 요청하실 수 있습니다.
